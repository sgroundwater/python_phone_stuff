# Gluebox Telephone Signal Automation Kit
# python_phone_stuff

This is a small collection of maker automation project software. It's a mix of code techniques taken from 
across the Intenet. This stuff might be helpful if you're connecting a bunch of stuff, especially Raspberry 
Pi, Arduino and Rotary Dial Telephones :-)

    - This code has no license. 
    - Much of this code originated from various sources. 
    - Protip : Use Audacity to generate two tracks, generate tones @ 350 and 440 Hz - Enjoy
    - See the whole project @ https://GLUEBOX.com/telephone_stuff
